Este es el juego de Triqui donde dos jugadores, X y O y despues de cada movimiento (click) de un jugador, se verifica si con este movimiento se ha ganado la partida y se manda un mensaje de felicitación o sino se reinicia el juego.


<< Inicio OVI Juego del Ahorcado >>

Turno del jugador: X

    Instrucciones del Juego:

  • 1. Dos jugadores, X y O
  • 2. Despues de cada movimiento (click) de un jugador, se verifica si con este movimiento se ha ganado la partida y se manda un mensaje de felicitación
  • 3. Si despues de un movimiento no se gana la partida, se cede el turno al otro jugador
  • 4. Actualizar en cada cambio de turno el mensaje en la parte inferior del tablero
  • 5. Solo se podra "marcar" una celda que NO ha sido previamente marcada.
  • 6.Cuando el tablero se ha completado sin identificarse un ganador, se reinicia el juego
