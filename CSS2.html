Implementación de un Archivo CSS

En la división del <head> estableceremos la ruta donde se encuentra ubicado estos archivos, en lo general siempre hay una carpeta Content donde se almacenan todos estos archivos .CSS

<head>
<link href= "Content/Style.css" rel="stylesheet" />
</head>

Aplicabilidad

En la hoja de estilo creamos todas las etiquetas que se le van a utilizar durante la creación del proyecto, debemos tener presente los colores a implementar, las imágenes, los tipos de tamaño y estilo de letras, recordemos que la extensión para guardar estos archivos debe ser .css

p {
    font-family: 'Poppins', sans-serif;
    font-size: 1.1em;
    font-weight: 300;
    line-height: 1.7em;
    color: #000000;
}


El HTML ya solo es escribir la etiqueta y el contenido que se requiere

  <p1> En este espacio escribimos el contenido deseado </p1>
